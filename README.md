# Webpack 2 Starter Kit

A starter kit for frontend developing with Webpack 2, ECMAScript 6, Handlebars templates, SCSS, Foundation for sites and separated data to JSON files.

## Installation

    npm install


## Developing

Kicks of local webserver with live reload and hot module replacement (HMR) [http://localhost:3000](http://localhost:3000)

    npm run dev


## Building

Builds your site in folder `dist`, minified and optimized

    npm run build
